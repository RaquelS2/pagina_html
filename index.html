Tag P

Em HTML, o elemento P é utilizado para definir um parágrafo. Ele serve para agrupar um bloco de texto e indicar ao navegador que este conteúdo deve ser apresentado como um parágrafo independente.

Para que serve?

Organização do texto: Divide o conteúdo em unidades menores e mais fáceis de ler.
Formatação: Os navegadores, por padrão, aplicam uma formatação básica aos parágrafos, como espaçamento entre linhas e indentação da primeira linha.
Semântica: Indica aos navegadores e mecanismos de busca que o conteúdo dentro das tags P é um parágrafo, o que ajuda na indexação e acessibilidade do seu site.

Tag I

O elemento HTML I coloca o texto em ítalico representando uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios.

Tags H1 ao H6

Os elementos HTML de cabeçalho da seção

Os elementos HTML H1 ao H6 representam seis níveis de título de seção. H1 é o nível de seção mais alto e H6 é o mais baixo.

Elementos de cabeçalho são implementados em seis níveis, H1 é o mais importante e H6 é o de menor importância. Um elemento de cabeçalho descreve brevemente o tópico da seção em que ele está. As informações de cabeçalho podem ser usadas por agentes de usuário, por exemplo, para construir uma tabela de conteúdos para um documento automaticamente.

Tag U

A tag U é uma das tags de formatação de texto disponíveis no HTML. Ela é usada para sublinhar o texto dentro de um elemento específico.

Tag Strong

No HTML é usada para dar ênfase forte a um trecho de texto. Por padrão, os navegadores exibem o conteúdo dentro dessa tag em negrito, mas o propósito principal dela é semântico, ou seja, indicar que aquele texto tem uma importância maior no contexto do conteúdo.

Tag Small

A tag small no HTML é usada para indicar texto que deve ser exibido em um tamanho menor em relação ao texto ao seu redor. Geralmente, ela é utilizada para informações secundárias, notas de rodapé.

Tag Mark

A tag mark no HTML é usada para destacar texto dentro de um conteúdo. O texto envolvido por essa tag geralmente aparece com um fundo amarelo (ou outro estilo, dependendo do CSS aplicado), simulando o efeito de um marcador de texto físico.

Tag blockquote

A tag blockquote no HTML é usada para indicar uma citação em bloco, ou seja, uma citação longa que geralmente ocupa mais de uma linha. Ela é semanticamente apropriada para destacar trechos de texto que foram retirados de outra fonte, como citações de livros, artigos ou discursos.

Tag sup

A tag sup no HTML é usada para exibir texto em sobrescrito, ou seja, ligeiramente acima da linha base do texto normal. É comumente utilizada para representar expoentes, números ordinais (como 1º, 2º), fórmulas matemáticas ou referências em textos acadêmicos.
